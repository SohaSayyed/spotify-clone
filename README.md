# Spotify UI Clone 

A pixel-accurate clone of the Spotify web player UI, built from scratch 
using HTML and CSS.

##  Built With
- HTML5
- CSS3 (Flexbox, Media Queries, Custom Range Input)
- Font Awesome 6 (icons)
- Google Fonts – Montserrat

##  Features
- Responsive sidebar with navigation and library section
- Sticky top navigation bar
- Music cards grid with hover effects
- Bottom music player bar with custom styled progress bar
- Responsive layout — sidebar hides on screens below 1000px

##  Project Structure
spotify-clone/
├── index.html
├── css_project.css
└── images/
├── card1img.jpeg ... card6img.jpeg
├── player_icon1.png ... player_icon5.png
└── (other icons)

##  How to Run Locally
1. Clone this repository
2. Open `index.html` in your browser
   — or use Live Server in VS Code for auto-refresh
